# UAS-PBO
SIPUDI (Aplikasi Peminjaman Buku Digital) adalah aplikasi desktop sederhana yang dibuat menggunakan bahasa pemrograman Python dengan menerapkan konsep Object-Oriented Programming (OOP). Aplikasi ini dirancang untuk membantu proses peminjaman dan pengembalian buku agar lebih rapi, terstruktur, dan mudah dikelola.
Dengan antarmuka grafis (GUI) berbasis Tkinter, pengguna dapat mengoperasikan aplikasi secara visual tanpa perlu berinteraksi langsung dengan kode program. Aplikasi ini cocok digunakan sebagai proyek pembelajaran OOP maupun sebagai simulasi sistem peminjaman buku sederhana.

Tata Cara Menggunakan Aplikasi
1. Menjalankan Aplikasi, Setelah aplikasi dibuka, pengguna akan melihat tampilan utama aplikasi peminjaman buku.
2. Memilih Buku, Pilih buku yang tersedia berdasarkan kategori atau daftar yang ditampilkan.
3. Masukkan data peminjam.
4. Pilih buku yang ingin dipinjam
5. Klik tombol Pinjam Buku untuk menyimpan data peminjaman.
6. Pengembalian Buku. Pilih buku yang ingin dikembalikan. Klik tombol Kembalikan Buku
7. Ekspor Data dengan klik tombol Ekspor CSV.
